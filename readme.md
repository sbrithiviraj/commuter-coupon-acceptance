# Project Title: 
Restaurant Coupon Acceptance Analysis for Commuters
# Description:
An exploratory data analysis on a UCI Machine Learning dataset and was collected via a survey on Amazon Mechanical Turk. Evaluating consumer responses to real-time, location based driving coupons depending on the time of dat, current weather, passenter types and coupon categoeris (less expensive restaurants (under \$20), coffee houses, carry out & take away, bar, and more expensive restaurants (\$20 - $50)). Highlight the differences between customers who did and did not accept the coupons.
